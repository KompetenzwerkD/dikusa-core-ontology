# Vocabularies

This folder contains skos-vocabularies used in the DIKUSA project in turtle-format.

## List of Vocabularies
* HAIT_DIKUSA_discrimination_experience.ttl is a list of categories of discrimination experiences used in the DIKUSA project of the Hannah Arendt Institute for Totalitarianism Studies (HAIT) concerned with experiences during displacement of Jewish women
* HAIT_DIKUSA_emancipation.ttl is a list of categories of emancipation used in the DIKUSA project of the Hannah Arendt Institute for Totalitarianism Studies (HAIT) concerned with experiences during displacement of Jewish women 
* HAIT_DIKUSA_emotions.ttl is a list of categories of emotions used in the DIKUSA project of the Hannah Arendt Institute for Totalitarianism Studies (HAIT) concerned with experiences during displacement of Jewish women
* werkd_country.ttl contains countries based on the country-category of GND
* werkd_country_extended.ttl contains additional metadata about the countries
* lido_genderActor.ttl lists genders based on LIDO gender actor
* hisco.ttl contains occupations based on the Historical International Standard of Classification of Occupations (HISCO)
